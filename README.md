🛡️ API Sentinel
API Sentinel is an advanced, automated API security scanner designed to identify critical vulnerabilities and misconfigurations in real-time. Built with a "Security-First" philosophy, it bridges the gap between complex vulnerability detection and developer-friendly remediation through an integrated AI-driven consultant.

🚀 Core Features
Active Threat Simulation: Performs real-time probes for Transport Layer Security (HTTPS), Rate Limiting, and Information Leakage.

Endpoint Auto-Discovery: Automated "Fuzzing" of hidden directories and sensitive paths (e.g., /.env, /admin).

Security Health Scoring: A deductive scoring engine that provides an instant posture assessment from 0-100.

AI-Powered Analyst: A built-in, asynchronous chatbot that provides "Simply Put" analogies and actionable code-fix snippets (Nginx, Apache, FastAPI).

Cyber-Grid UI: A high-fidelity, ethical-hacking-themed dashboard featuring glassmorphism and real-time terminal logs.

🛠️ Technology Stack
Backend: Python 3, FastAPI, Uvicorn, Requests, Asyncio.

Frontend: HTML5, Tailwind CSS, JavaScript (Fetch API).

Security Logic: Inspired by OWASP API Top 10 standards.
